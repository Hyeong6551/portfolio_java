# JAVA 도서관리 프로그램
Java 도서관리 프로그램


## 프로젝트 소개
관리자는 도서를 추가/변경/삭제
사용자는 대출/반납을 할 수 있는 프로그램


## 개발 기간
- 2025.01.09 ~ 2025.02-14


### 개발 환경
- `java 21`
- `JDK 21`
- IDE : Eclipse
- Database : Oracle (11g)
- jdbc : [ojdbc17.jar](https://download.oracle.com/otn-pub/otn_software/jdbc/237/ojdbc17.jar)


## 주요 기능
### 관리자 페이지
C : 책 추가 / R : 도서 목록 조회 / U : 책 변경 / D : 책 삭제


### 사용자 페이지
C : 책 대출 / R : 내 도서목록 조회 / U : X / D : 책 반납
